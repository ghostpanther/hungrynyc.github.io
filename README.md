HungryNYC
==========

For the hungry and indecisive New Yorker.

## HungryNYC, the App

HungryNYC was built on Ruby on Rails, JavaScript, jQuery, HTML5, CSS3, PostgreSQL, Yelp API, and Google Maps API.
